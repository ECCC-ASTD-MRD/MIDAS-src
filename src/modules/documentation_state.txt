TO VIEW THE AUTOMATIC DOCUMENTATION RESULTING FROM A SINGLE FILE:
=================================================================
- cd scripts/sphinx
- edit build_html.sh:
    - set:  module_filelist="../../src/modules/<your file name>.f90"
./build_html.sh

The result will appear here (as html files):  ~/public_html/midas_sphinx

View the result here:  http://goc-dx.science.gc.ca/<your home>/midas_sphinx/
     (e.g. http://goc-dx.science.gc.ca/~jbl001/midas_sphinx/
